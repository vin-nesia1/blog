Apa Itu QR Code WiFi?

QR code WiFi adalah jenis Quick Response code khusus yang menyimpan informasi jaringan WiFi, termasuk nama SSID, password, dan jenis keamanan. Ketika dipindai dengan smartphone, perangkat akan otomatis terhubung ke jaringan tanpa memerlukan input manual dari pengguna.

Format QR Code WiFi

QR code WiFi menggunakan format string khusus yang mengikuti standar internasional. Berikut contoh format yang digunakan:

WIFI:T:WPA;S:NamaWiFi;P:PasswordWiFi;H:false;;

Dimana:

‚Ä¢ T = Type (WPA, WEP, atau kosong untuk open network)

‚Ä¢ S = SSID (nama jaringan WiFi)

‚Ä¢ P = Password (kata sandi jaringan)

‚Ä¢ H = Hidden (true/false untuk jaringan tersembunyi)

Cara Scan QR Code WiFi di Berbagai Perangkat

Android (8.1+)

Smartphone Android versi 8.1 ke atas sudah mendukung WiFi QR code secara native. Buka Settings > WiFi > tap ikon QR atau "Add Network" > pilih "Scan QR Code".

iPhone (iOS 11+)

iPhone dapat scan QR WiFi melalui kamera bawaan atau Control Center. Arahkan kamera ke QR code, notification akan muncul untuk join network.

Perangkat Lama

Untuk perangkat yang belum support native, gunakan aplikasi third-party seperti WiFi QR Code Scanner atau QR Code Reader.

Troubleshooting QR Code WiFi

QR Code Tidak Terbaca

Pastikan QR code dalam kondisi bersih, tidak buram, dan memiliki kontras yang cukup. Coba scan dari jarak dan sudut yang berbeda.

Gagal Koneksi Setelah Scan

Verifikasi kembali SSID dan password yang digunakan saat membuat QR. Periksa juga jenis keamanan jaringan sudah sesuai dengan setting router.

Smartphone Tidak Support

Download aplikasi WiFi QR Scanner dari Play Store atau App Store sebagai alternatif untuk perangkat yang belum mendukung fitur native.
