Apa Itu vCard dan QR Code vCard?

vCard adalah format standar untuk menyimpan dan bertukar informasi kontak digital. Format ini mendukung berbagai data seperti nama, nomor telepon, email, alamat, website, hingga foto profil dalam satu file yang compact.

QR code vCard adalah Quick Response code yang menyimpan data vCard sehingga memungkinkan orang lain mengimpor informasi kontak Anda langsung ke smartphone mereka dengan sekali scan.

Format vCard Standard

vCard menggunakan format text-based yang mengikuti standar RFC 6350. Berikut contoh struktur vCard sederhana:

BEGIN:VCARD VERSION:3.0 FN:John Doe N:Doe;John;;; ORG:Tech Company TITLE:Marketing Manager TEL;TYPE=CELL:+62812345678 TEL;TYPE=WORK:+62215551234 EMAIL;TYPE=WORK:john@company.com URL:https://www.company.com ADR;TYPE=WORK:;;Jl. Sudirman 123;Jakarta;;12345;Indonesia NOTE:Digital Marketing Specialist END:VCARD

Troubleshooting Common Issues

QR Code Tidak Terbaca

Pastikan QR code memiliki resolusi minimal 300 DPI untuk cetakan dan kontras yang cukup antara foreground dan background.

Beberapa Field Tidak Muncul

Tidak semua aplikasi kontak mendukung semua field vCard. Field standar seperti nama, telepon, dan email umumnya selalu support.

Foto Tidak Tampil

Gunakan foto dengan ukuran maksimal 100KB dan format JPEG untuk kompatibilitas maksimal.
